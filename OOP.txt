Мы имеем шахматные фигуры (piece): пешки (pawn) и ферзи (queen).
Каждая фигура имеет:
1. Цвет - строка. Значение 'black' или 'white'.
2. Свое положение на доске заданное в виде текстовой строки. Например "E2".
3. Цель для хода - строка. Например "E4".
4. Возможности для хода. Список строк. Например ["E3", "E4"]
5. Возможности для удара. Список строк. Например ["D3", "F3"]
6. Ход на некоторую клетку, в случае, если она пустая или бой фигуры.


Имеем шахматную доску, в которой хранится список фигур по типам.
Доска имеет функциональность:
1. По координате("E2") получить фигуру, которая там стоит.
2. Убрать фигуру с доски (по конкретной координате).
3. Добавить фигуру на доску (по конкретной координате).

Реализовать данное задание с помощью ООП
